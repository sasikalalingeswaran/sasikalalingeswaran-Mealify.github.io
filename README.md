Mealify - Smart Meal Planner & Recipe Finder

Overview

Mealify is a React-based web application that helps users discover recipes, save favorites, plan weekly meals, manage grocery lists, and get nutrition-related suggestions through an integrated chatbot.

Features

Search meals by name
Browse meal categories
View detailed recipes and ingredients
Add meals to Favorites
Weekly Meal Planner
Grocery List Management
Nutrition Chatbot
Responsive Design
Local Storage Support

Technologies Used

React.js
JavaScript
CSS3
React Hooks
Local Storage
TheMealDB API



Installation

cd mealify
npm install
npm run dev
